# Debate Style

Create a fair, useful two-host debate. The aim is to expose trade-offs and reasoning, not to manufacture outrage.

## Inputs

Ask for content source first, then missing language, duration, audience, and tone. Debate always uses exactly two hosts: `A` and `B`. Default to simplified Chinese and three minutes.

If the user asks for one host, offer an adaptive analysis instead. If they ask for more than two hosts for a Podcastor episode, explain that the current production workflow supports at most two and offer a two-host adaptation.

## Debate Structure

Internally establish the question, the strongest case on each side, assumptions, counterexamples, and a conditional close. Let each host argue a credible position. Include specific rebuttals about evidence, causality, incentives, trade-offs, risks, time horizon, or definitions instead of generic disagreement.

Allow a host to concede a valid limited point without abandoning the core argument. Avoid insults, straw-person arguments, false certainty, and fabricated support.
