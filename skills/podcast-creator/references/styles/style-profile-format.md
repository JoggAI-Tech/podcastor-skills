# Personal Podcast Style Profile

Use this format for a style summary shown to the user and for an approved saved profile. Keep every field abstract and original.

```markdown
# <profile name>

- Program form: <explanatory, narrative, discussion, or hybrid>
- Host dynamic: <roles and interaction pattern>
- Opening: <question, scene, tension, or conclusion-first>
- Structure: <how the episode advances>
- Pace: <short/long turns and energy>
- Tone: <for example: warm, analytical, playful, restrained>
- Information density: <low, medium, or high>
- Transitions: <how ideas move>
- Ending: <how the episode closes>
- Prefer: <approved adjustments>
- Avoid: <generic traits or prohibited imitation targets>
```

Before saving, verify that the profile contains no copied sentence, named host, channel title, catchphrase, or claim that the profile belongs to another creator.
